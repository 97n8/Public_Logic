Quick Setup — Google Workspace (Sutton)

Goal: Be live in under 30 minutes. These steps are human‑readable and safe to delegate.

  1. Create shared inboxes: rao@town.sutton.ma.us and clerk-intake@town.sutton.ma.us.
  2. Add Gmail labels: vault/intake, vault/hold, vault/published.
  3. Create a Shared Drive: Sutton_VAULTCLERK with folder /intake_scans (300 dpi PDFs).
  4. Create a Google Sheet: Phone_311_Log with headers datetime,caller,subject,location,notes.
  5. Point Polimorphic form endpoint to /intake.

Apps Script: Label Creator (optional)

In Gmail, open Settings → Advanced → Apps Script, create a script, and paste this:

function createVaultLabels() {
  const labels = ["vault/intake","vault/hold","vault/published"];
  labels.forEach(name => GmailApp.createLabel(name));
  Logger.log("Labels ready.");
}
