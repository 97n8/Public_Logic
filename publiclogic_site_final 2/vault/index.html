VAULT module shell

This page lists example modules and towns for the VAULT family. Each module view is driven by a JSON file in the /data folder and can be wired to SharePoint and Power Automate.

VAULTCLERK | Sutton, MA

Example shell for a Clerk module. The public view pulls from data/vaultclerk-sutton.json and renders a simple table of metrics and notes.

  • Module code: VAULTCLERK
  • Town: Sutton, Massachusetts
  • Status: Sample only

How to add a new module

To add a new module and town, create a JSON file in /data following the standard schema and then link to the generic module.html view with the right query parameters.
