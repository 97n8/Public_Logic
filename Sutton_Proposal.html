Plain Talk

If a resident can't submit a request without calling, if staff can't see what's due when, and if we can't export history on demand—it isn’t better connections. SuttonCONNECT fixes those three things first, then connects everything else: resident intake → staff workflows → compliance exports → grant reporting → next year's funding. Massachusetts rules are built in: Public Records clocks (10-day ack, 25-day max), OML posting deadlines (48h), dog license renewals, and retention schedules. Immutable audit logs protect staff if deadlines slip.

Why Sutton Right Now

When someone retires or is out, institutional knowledge often walks out the door. SuttonCONNECT is designed around succession resilience using the VAULT Framework—preserving know-how as queues, checklists, and case histories.

VAULT, CLERK & FIX

VAULT (patent-pending) ensures verified automation, unified logic, and town-owned processes. CLERK organizes licensing, vital records, PRR, agendas, and elections. FIX standardizes DPW, facilities, and maintenance workflows. Together, they provide a compliance shield and staff clarity.

What Changes for Sutton

Daily Work

  • Public Records clock starts automatically, with countdown in dashboards
  • DPW requests include photo + map pin
  • No close without “after” documentation
  • Supervisors see timers and can reassign early

Records

  • Quarterly exports land in Shared Drive with manifest + checksums
  • Case exports available on demand (PDF + JSON + attachments)
  • Agenda postings timestamped with receipts
