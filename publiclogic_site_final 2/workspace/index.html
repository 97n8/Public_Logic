This is where we start and track projects. Everything begins in Universal Intake, files are created in SharePoint under the right folders, and routing is handled by the flows.

Start new project Opens Universal Intake. The form creates the SharePoint folders and files for you.

Use the button to start. Once the intake is submitted, the system builds the folders, drops the right files into SharePoint, and sends the project to the Processing board. When work is done, it moves to Archive with links back to the town and module.

Boards

  • Intake board
    New towns, ideas, and requests.
  • Processing board
    Items we are actively building or wiring.
  • Archive board
    Completed work recorded by the system.

How this is meant to work

  • Start projects from this page using Universal Intake.
  • Let the automations create and name folders and files in SharePoint.
  • Work from the Processing board instead of going into file trees.
  • When a project is done, move it to Archive and keep the links updated.
