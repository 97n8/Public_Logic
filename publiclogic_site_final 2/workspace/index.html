PublicLogic workspace

This is our working home screen. Start new projects here and track them as they move from intake, to active work, to archive. SharePoint and the automations handle the folder tree and files in the background.

Start new project (Intake form) Opens the live PublicLogic intake form. Submitting creates a new item in the intake SharePoint list and starts the project flow for municipal builds and research work.
1 Intake
2 Processing
3 Archive

Step 1: Use the intake form for any town, module, or research initiative. Step 2: The flow writes to the intake SharePoint list and builds the folder structure and starter files. Step 3: The project appears on the Processing board. When work is complete, it moves to Archive with links back to the modules and files.

Boards

  • Intake form
    One form for municipal systems work and research initiatives, including PublicInsight and EFRI.
  • Processing board
    Projects we are actively building, wiring, or delivering.
  • Archive board
    Completed projects with links back to their files and modules.

How to use this page

  • Always start a new town, module, or research initiative with the intake button.
  • Work from the Processing board instead of browsing deep into file trees.
  • Keep Archive updated so we can see what has been delivered and where it lives.
  • If structure looks off, flag it. Do not rename folders or files by hand.
